Recipiece

Recipiece is a simple online cookbook. Users can store their recipes as either public or private and can easily access them through a simple grid view. There are many more planned features. Recipiece uses Angular for the application front-end and PostgreSQL/nodeJS for the backend. More Information

PXV

PXV is an RPG that I am currently using to learn Unity. Scripts are written in C#, and maps are made in Tiled. I've wanted to learn Unity for a long time, and I love (J)RPG games. More Information

Coffee Shop

This is a simple coffee shop template-esque website. I like clean and simple layouts, so I elected to use UI Kit for its suite of components and its clean look and feel. I made this as an "I want to do something nice and easy" project; it's based off of the current website for my favorite coffee shop. More Information
